\# Contribuindo

&nbsp; 

Contribuições são bem-vindas!

&nbsp; 

1\. Clone o repositório

&nbsp; 

```bash

git clone https://github.com/vfranca/mtcli.git

cd mtcli

```

&nbsp; 

2\. Instale dependências de desenvolvimento

&nbsp; 

```bash

pip install -e ".\[dev]"

```

&nbsp; 

3\. Testes

&nbsp; 

```bash

pytest

```

&nbsp; 

4\. Estilo

&nbsp; 

\- Use black, pydocstyle e isort
